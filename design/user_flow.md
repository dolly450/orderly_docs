# 1. Διαδρομή Πελάτη (User Flow)
Η εμπειρία του πελάτη από το σκανάρισμα του QR code μέχρι την παραλαβή της παραγγελίας. Η τρέχουσα ροή είναι cloud-first web flow, με offline/local fallback να παραμένει future-phase ιδέα.

### Οπτικοποίηση

```mermaid
graph TD
    A[Έναρξη: Σκανάρισμα QR code στο τραπέζι ή stand] --> B[Web Flow / Start Screen]
    B --> C[Πλοήγηση στο μενού και επιλογή προϊόντων]
    C --> D[Cart / Checkout]
    D --> E[Αποστολή στο Staff Dashboard]
    E --> F[Ο πελάτης λαμβάνει αριθμό παραγγελίας και εκτιμώμενο χρόνο]
    F --> G{Έτοιμη η παραγγελία}
    G -- Ναι --> H[Status Update / Ειδοποίηση]
    H --> I[Παραλαβή στο Stand και ταυτοποίηση]
    I --> J[Ολοκλήρωση παραγγελίας]
```
